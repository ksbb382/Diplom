# Справочник оформления — ВКР Рыжкина (эталон кафедры АИТП РГРТУ)

Источник: `ВКР Рыжкин А. А. 237.docx`. Все данные извлечены из XML-структуры документа без догадок.

> **Внимание (важная находка):** в `theme1.xml` Рыжкина прописан Calibri как тема, и в `docDefaults` стоит `asciiTheme="minorHAnsi"`. Но! **На каждом ране** документа стоит явный `<w:rFonts w:ascii="Liberation Serif"/>` — то есть Рыжкин (или его автор-нейросеть) **переопределил шрифт прямо в ранах**. Поэтому визуально всё работает как Liberation Serif. Это и нужно сделать у нас.

---

## ЕДИНИЦЫ ИЗМЕРЕНИЯ В OOXML

- **Размер шрифта:** в half-points. `sz=28` = 14pt, `sz=32` = 16pt, `sz=40` = 20pt.
- **Межстрочный интервал:** `line` в twentieths-of-a-point. `line=240` = 1.0, `line=276` = 1.15, `line=360` = 1.5.
- **Отступы:** `firstLine="709"` = 1.25 см (1 см ≈ 567 twentieths).

## КЛЮЧЕВЫЕ ВЫВОДЫ ДЛЯ НАШЕГО ФАЙЛА

1. **Шрифт явно прописан в каждом ране** — нельзя полагаться на `docDefaults` или тему. Нужно проставить Liberation Serif в каждом `<w:rFonts>`.
2. **Размер 14 pt = `sz=28`** в half-points. Заголовки — больше (см. таблицу выше).
3. **Межстрочный = 1.5 (`line=360`)** для тела документа. Для шапки титула — 1.0 (`line=240`).
4. **Отступ первой строки = 1.25 см (`firstLine=709`)** — только для тела абзацев в аннотации, введении, главах.
5. **Выравнивание тела = both (по ширине)**, заголовков — center, шапок и подписей — по контексту.
6. **Тема `theme1.xml`** не критична, главное — явные ссылки в ранах.

## АЛГОРИТМ ИСПРАВЛЕНИЯ НАШЕГО ФАЙЛА

1. **Не трогать `theme1.xml` Рыжкина** (либо привести к Liberation Serif — для безопасности).
2. **В каждом `<w:r>` нашего docx**:
   - Установить `<w:rFonts ascii="Liberation Serif" hAnsi="Liberation Serif" cs="Liberation Serif" eastAsia="Liberation Serif"/>`
   - Удалить все theme-attrs (`asciiTheme`, `hAnsiTheme`, `cstheme`).
   - Установить `<w:sz w:val="28"/>` (14pt) — кроме заголовков (там 32 для Heading 1, 28 для Heading 2/3).
3. **В каждом `<w:p>` для тела диплома** (главы, введение, заключение, аннотация):
   - `<w:pPr><w:spacing line="360" lineRule="auto"/><w:ind firstLine="709"/><w:jc val="both"/></w:pPr>`
4. **Заголовки (Heading 1/2/3)** — center, без firstLine, line=360, жирные.
5. **`docDefaults` в styles.xml**:
   - rFonts: Liberation Serif (всё)
   - sz: 28 (14pt)
   - spacing line=360 (1.5)
